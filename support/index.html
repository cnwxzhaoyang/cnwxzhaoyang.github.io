ASeek Support

ASeek is a lightweight macOS app for local file, folder, and application search. This page provides support information for App Store users.

Common Topics

Add folders for indexing

Open ASeek Settings, go to Search, then choose the folders you want ASeek to index. macOS will ask you to authorize folder access.

Change the global shortcut

Open Settings, go to General, then record a new shortcut. If another app already uses the shortcut, choose a different one.

Manage scan permissions

ASeek indexes only folders you authorize. Remove folders from scan roots when you no longer want them included.

Reset preferences

Open Settings and use Reset preferences to restore shortcuts, appearance, preview, and ignored folders to defaults.

Search Syntax

Use advanced operators when a simple keyword returns too many results. Operators can be combined in one query.

ext:pdfLimit results to one extension
path:downloadsNarrow matches to a path fragment
name:reportSearch only within filenames
folder: file:Restrict results to folders or files
!draftExclude unwanted keywords
"annual report"Treat a spaced phrase as one unit
*.rsMatch all Rust files
app*.vueNames starting with app and ending with .vue
ext:rs path:srcOnly Rust files under the src path
path:components !test ext:vueVue files in component paths while excluding tests
folder: name:src*Folders whose names start with src
dir: path:project *demo*Directories under project paths that also match demo

Matching covers names, paths, app aliases, pinyin, and initials. name: searches names only, while path: searches paths only.
