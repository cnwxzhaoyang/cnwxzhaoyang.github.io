ASeek

Fast local file search for Mac.

ASeek helps you find files, folders, and apps from a compact keyboard-first search window. Choose the folders you want to index, search instantly, and keep your file data on your Mac.

Technical Support Privacy Policy

What ASeek does

Local file search

Build a local index for selected folders and quickly search file names, folders, and applications without sending your index to a server.

Keyboard-first access

Open the search panel with a global shortcut, type, navigate results, and open files without breaking focus.

Flexible scan roots

Add only the folders you want ASeek to scan. macOS folder permissions are requested explicitly and can be changed later.

Useful previews

Review key file details and preview supported items before opening them, so search results are easier to compare.

Highlights

  • Designed for macOS with a compact search window and menu bar access.
  • Local-first indexing for file names, paths, recent activity, and ranking.
  • Customizable global shortcut, appearance, language, preview behavior, and ignored folders.
  • Sandbox-friendly folder authorization using macOS security-scoped bookmarks.
  • No account required and no upload of file names, paths, contents, or search queries.

Advanced Search

ASeek supports Everything-style search operators for narrowing results without leaving the search box.

ext:pdfLimit results to one extension
path:downloadsNarrow matches to a path fragment
name:reportSearch only within filenames
folder: file:Restrict results to folders or files
!draftExclude unwanted keywords
"annual report"Treat a spaced phrase as one unit

* matches any length, ? matches one character, spaces act as AND, and filters can be combined.

ext:rs path:srcRust files under a src path
invoice !draftInvoices excluding draft matches
path:components !test ext:vueVue components excluding tests
folder: name:src*Folders whose names start with src

主要功能

本地文件搜索

为你选择的目录建立本地索引，快速搜索文件、文件夹和应用，索引数据不上传服务器。

快捷键唤起

通过全局快捷键打开搜索面板，输入关键词后即可用键盘浏览结果并打开文件。

目录权限可控

只扫描你主动添加并授权的目录，不需要索引时可以随时从扫描路径中移除。

预览与结果对比

搜索结果会展示文件信息，并支持预览部分文件，减少打开错误文件的次数。

高级搜索

ASeek 支持类似 Everything 的搜索语法，可直接在搜索框中用条件缩小结果范围。

ext:pdf只看指定扩展名
path:downloads把结果限制在某段路径内
name:report只在文件名范围内查找
folder: file:限定结果类型为目录或文件
!draft排除不想看到的关键词
"annual report"把带空格的短语视为一个整体

* 匹配任意长度字符，? 匹配任意单个字符， 空格分隔多个条件并按 AND 处理，多个过滤条件可以组合使用。

ext:rs path:src只看 src 路径下的 Rust 文件
invoice !draft包含 invoice，但排除 draft
path:components !test ext:vue组件目录下的 Vue 文件，并排除测试相关项
folder: name:src*只搜索名称以 src 开头的文件夹
