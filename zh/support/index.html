ASeek 支持

ASeek 是一款 macOS 本地文件、文件夹和应用搜索工具。本页面为 App Store 用户提供支持信息。

常见主题

添加扫描目录

打开 ASeek 设置，进入“搜索”，选择需要索引的目录。macOS 会请求目录访问授权。

修改全局快捷键

打开“通用”设置并录制新的快捷键。如果快捷键已被其他应用占用，请换一个组合。

管理目录权限

ASeek 只索引你授权的目录。不再需要索引时，可以从扫描路径中移除对应目录。

恢复默认偏好设置

在设置中使用“恢复默认偏好设置”，可重置快捷键、外观、预览和忽略目录等偏好。

快捷键

CommandK打开或聚焦 ASeek 搜索
↑↓切换搜索结果
Enter打开选中的结果
CommandEnter在 Finder 中显示选中结果
CommandShiftC复制选中文件路径
Esc隐藏搜索窗口

搜索语法

当普通关键词返回结果太多时，可以使用高级搜索条件；多个条件可以组合在同一个查询中。

*.rs匹配所有 Rust 文件
app*.vue匹配以 app 开头、以 .vue 结尾的名称
ext:rs path:src只看 src 路径下的 Rust 文件
path:components !test ext:vue组件目录下的 Vue 文件，并排除测试相关项
folder: name:src*只搜索名称以 src 开头的文件夹
dir: path:project *demo*只搜索项目路径下并命中 demo 的目录
